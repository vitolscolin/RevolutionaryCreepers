# RevolutionaryCreepers

A full-stack risk simulation demo built with React and Node/Express.

## Scripts

- `npm install` installs root, client, and server dependencies
- `npm run dev` starts the frontend and backend together
- `npm run build` builds the React frontend
- `npm run start` runs the Express server

## App Overview

The app includes:

- A React frontend with sliders and dropdowns
- A Node/Express backend
- A `/api/simulate-risk` endpoint for risk simulation
- Before and after risk score comparisons
